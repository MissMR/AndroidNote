# Linux常用命令

## Linux文件类型
       类型          文件开头
   1. 普通文件 ----- -
   2. 目录文件 ----- d
   3. 链接文件 ----- l
   4. 设备文件 ----- b/c
   5. 管道    ----- p
   6. 套接字  ----- s
## 文件查看命令
    1. ls 查看目录下的文件名
    2. ls -l 查看目录下的文件详情，名称和类型等等。
    3. 查看当前目录 pwd
    4. 查看家目录 echo $HOME
## 用户操作命令
    1. useradd 添加用户
    2. userdel 删除用户
    3. [sudo] + passwd  + 用户名 设置密码(sudo 表示给root用户修改密码)
    4. who 显示登录到系统的所有用户信息
    5. su - 用户名 切换用户
## 系统管理命令
    1. ps -aux 显示进程表
    2. shutdown -h/-r 关闭\重启
    3. uptime 运行时间
    4. clear 清屏
